fn run( self, message_bus: &mut MessageBus<Self> ) -> impl Future<Output = Result<(), Self::Error>> + Send

Runs this tasks. The task can use the provided MessageBus for communication. When the MessageBus has no more messages to be consumed, the task should exit without errors.
