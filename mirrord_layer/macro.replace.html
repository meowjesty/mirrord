Macro mirrord_layer::replace

source · 
macro_rules! replace {
    ($hook_manager:expr, $func:expr, $detour_function:expr, $detour_type:ty, $hook_fn:expr) => { ... };
}
Expand description

Replaces the $func libc function, with the equivalent hook $detour_function, by calling HookManager::hook_export_or_any.

Parameters

  • $hook_manager: a valid HookManager instance that is used to replace the libc function;

  • $func: the function we want to replace;

  • $detour_function: one of our detour functions;

  • $detour_type: the type alias that was generated by hook_fn for this function type;

  • $hook_fn: stores the original function pointer as a HookFn that is created by hook_fn.

Examples

  • Replacing libc::close with close_detour:
unsafe {
    replace!(&mut hook_manager, "close", close_detour, FnClose, FN_CLOSE);
}
