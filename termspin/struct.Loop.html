Run the loop with the given callback.

Example


let l = Loop::new(
    Duration::from_millis(100),
    Line::new(spinner::from_iter([r"\", "|", "/"]))
);

// Run the loop while blocking the current thread.
l.run(|out| print!("{out}"));

pub fn auto_stop(&self, stop: bool)

Exit the running loop if only one instance of the loop exists, defaults to true.

It is useful when you spawn the spawn the loop on a separate thread that should exit when all handles to it go out of scope.
